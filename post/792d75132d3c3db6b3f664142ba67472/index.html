我常用的Firefox扩展清单

2009-10-02 15:21

曾经接触Firefox很多次，但是爱上它是最近几个月的事情。Firefox这个浏览器，本身功能并不强大，他的强大之处在于扩展性强，有超过5000个的扩展供用户选择，多个扩展组合在一起，就诞生了一个属于你的强大浏览器。

Adblock Plus

很强大的一个扩展，可以让你摆脱网站中那烦人的广告，使用也很简单，对着你不想看的广告点击右键，选择屏蔽就可以了，下面看看我用该屏蔽《电影天堂》网站的广告效果。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/1865

Brief

操作简单的一款订阅扩展，加入订阅只需要把订阅页面拖拽到订阅标签里面就可以了。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/4578

Easy DragToGo

轻松的选取图片、文字、链接从新的标签中打开，支持手势，比如选中一段文字，然后向上拖拽就是使用百度搜索，向下拖拽就是使用 Google 搜索。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/6639

Firebug

很强大的一款调试工具，比如调试HTML、JavaScript、CSS等，说白了就是一款WEB开发工具，深受WEB开发人员的喜爱。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/1843

FireCookie

基于 Firebug 扩展的一个插件，用于查看和管理 Cookie ，非常好用，必须安装 Firebug 才能使用。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/6683

FireGestures

这是本人最喜爱的扩展之一，有了他，所有操作都能使用鼠标手势来控制，比如打开一个新的标签、关闭标签、刷新、恢复关闭页面等等，更简单的说，就是给浏览器部分功能加上一个快捷方式。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/6366

Flagfox

在地址栏显示当前页面服务器所在的国家国旗。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/5791

FlashGot

能够让你的 Firefox 调用外部下载工具，比如 FalshGet 、迅雷、QQ旋风等。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/220

Goo.gl Lite

Google提供的短址服务，点击按钮后直接把当前页面的短址复制到剪切板，没有任何提示。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/55308

Greasemonkey

面向 Firefox 用户的脚本管理器，点击这里查看我在用的 Greasemonkey 脚本。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/748

Google Gears

说真的，我也不知道这个是干什么的，是因为在WP后台看到说可以加速才安装上去的，不过我没发现有加速的迹象。

下载地址：http://gears.google.com/

Hide Menubar

可以把菜单栏隐藏起来，使得Firefox的界面更加简洁，需要使用的时候按ALT键就会出现。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/4762

IE Tab

让 Firefox 拥有两心脏，一般用于上支付宝、淘宝、银行等只支持IE的网站。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/1419

IE Tab 2

升级3.6以后，IE Tab一直没有更新到兼容3.6的版本，无奈，网上找到一个IE Tab 2，效果一样，就连设置界面也是一样的。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/92382

IE Tab Plus

基于 IETab 开发的增强版本，除具备 IETab 的全部功能外，还可以在 IE 引擎中用 Adblock Plus 过滤广告，以及同步 Cookie 使得切换到 IE 引擎时不需要重新登录。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/10909

InFormEnter

这个插件对于喜欢闲逛博客的人来说非常有用，每次想评论点什么，就不再需要手动输入昵称、Emial、网站等信息了，直接右键选择你要输入的信息就OK，具体介绍可以看这里。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/673

KeyConfig

最大化的扩充或者关闭 Firefox 的快捷键。Firefox 功能多了，快捷键也多，总是不小心打开下载、历史等快捷键，严重的影响了正常使用。寻找了N久，终于发现一款不错的插件，这款插件在官方的插件库是找不到的，因为官方不承认该插件，但是并不影响他的用户群，好东西是不会受到任何影响的。

下载地址：http://mozilla.dorando.at/keyconfig.xpi

Menu Editor

如果你觉得你的菜单栏、右键等菜单弹出的东西太多，这个扩展可以帮你简化，想要的就留下，不想要的就移除，大大的美化和简化了样式。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/710

Screengrab

截图工具，可以全页截图、窗口截图、浏览器截图、部分截图，可以保存为PNG和JPG格式。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/1146

StrataBuddy

美化firefox的窗口，配合下面的一个主题非常漂亮。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/14762

Strata40

这其实不是一个扩展工具，只是一个firefox的主题，但是这个主题配合上面那个插件才能发挥完美的效果。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/14284

Tab Mix Lite CE

可辅助 Firefox 的标签式浏览功能 ，喜欢简洁明了的轻度标签的人可以使用该扩展。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/12391

Tab Utilities

比 Tab Mix Lite CE 更加强大的标签扩展。

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/59961

Xmarks

用来在多台计算机中同步你的书签。最近Xmarks被墙了，但这并不影响我的书签同步，因为我发现了另外一个功能，就是可以把书签上传到自己的FTP中保存，这样是不是很方便？

下载地址：https://addons.mozilla.org/zh-CN/firefox/addon/2410

狐嘀

无需打开嘀咕的网站，直接在 Firefox 中发送和接收嘀咕信息。

下载地址：http://www.digufox.com/

以上就是我用的扩展了，还有一些QQMAIL、迅雷等软件安装的就不说了。很多人说 Firefox 不好用、垃圾，那是因为你们根本没有深入了解 Firefox ，什么扩展都不安装，我也觉得 Firefox 是个垃圾，因为没装扩展的 Firefox 我根本用不下去，还不如用IE。但是有了自己喜欢、合适的扩展就不同了，因为它们完全可以组合出一个属于你自己的浏览器。
