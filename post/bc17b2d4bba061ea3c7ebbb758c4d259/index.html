狗日的VS平台

2011-02-13 游戏
移动设备上继续阅读
目录

  2. 事件二：外挂投诉

VS竞技游戏对战平台（以下简称“狗日的”或“VS”）是国内比较火的一个游戏平台，大部分魔兽真三、DOTA的爱好者聚集在这里游戏。本人玩这个平台也有三四年历史了，骂他也有一两年的历史，我对这狗日的总结就是：没信用，不顾玩家感受的平台。我有根据，我没乱说。

事件一：VS三国活跃值兑换

VS三国是VS平台旗下的一款游戏，具体介绍看百度百科，VS的用户群大部分都是DOTA、真三玩家，VS也是一款类似的游戏。为了吸引DOTA、真三玩家去玩VS，狗日的推出了很多很多的活动，其中就有一个系列活动《VS三国新手5重礼》，第五个活动就是赚活跃值得永久VIP，1500活跃值可以兑换1天的VIP，上面还特别注明了，现在是活动期间，原3000活跃值兑换1天VIP，现在只要1500，活动过期后恢复3000。嗯，好吧，我无所谓，3000也比较合理。

我玩啊玩，玩啊玩，有近50000的活跃值了，可以换半个多月的VIP，可是有次我需要兑换的时候，这狗日的居然提示说：系统升级中，暂时不能兑换（后来我同学告诉我，这个提示已经快有一个星期了）。又等了几天，发现昨天可以兑换了，老子仔细一看，兑换一天VIP不是1500活跃值，也不是3000活跃值，是他妈的6000活跃值。

我艹你大爷的，你玩不起就别搞什么鬼活动啊，原本3000，怎么变6000了？系统升级？升你M个逼。

事件二：外挂投诉

游戏总是不公平的，外挂这玩意是个游戏它都会伴随着，为了给玩家一个公平的游戏环境，反外挂是每个游戏必须要有的功能模块。狗日的也有，真的，不信你去下一个几年前的外挂，绝对会被封杀掉，你再去下一个近两年的外挂，绝对不会被封杀掉，我说你反个外挂有这么难吗，兄弟？再说投诉吧，曾今我被一个外挂弄得实在是火大了，想去投诉，进去论坛投诉板块一看，完了，木有希望了。投诉分两种，一种是VIP投诉，一种是普通投诉，常常的帖子列表中，大部分回复都是 0 ，经常逛论坛的应该明白，这是没人回复，也就是说，管理员根本都不管的。你这投诉板块是吃屎的啊？投诉就投诉，还分VIP和普通。至少我前年的普通投诉，一直到现在还没回复或者有短消息通知。

关于投诉的除法，我以前听到过一种说法，如果玩家开了挂，被发现，会被封号，游戏中来说，这算比较严重的惩罚了。可在狗日的平台这里却不是，为什么？这狗日的平台道具商城中居然还有解除封号的道具买，真TM佩服这狗日的，想钱想疯了吧？

事件三：转盘活动

不知道什么时候的活动，反正也是弄了几天就没弄了，出现404错误。不是偶然，而是已经几天都没恢复，404，大家都懂的。

给大家看一个论坛的地址，很牛逼的路径：[http://bbs.vsa.com.cn/Discuz!/index.php]，真特么的专业啊，discuz都留着，版权意识顶顶的，这点要向狗日的敬礼。

还有，现在的网站都是登陆帐户以后，全站通行。可是狗日的为用户想的很多，为了防止我们忘记自己的帐号密码，这狗日的每个二级域名或者域名目录下的网站都不是通用的，这样确实能增强用户对帐户密码的记忆能力，想的太TM周到了。

还说一个，最后一个。找回密码，大家都应该知道密保问答是干什么的吧？就是你忘记了密码，然后提示你当初填写的问题，从而可以让你找回密码。可这个狗日的又考验我们的记忆了，找回密码的时候，居然还会问我，密保问答的问题是什么，我日你妹。

总之呢，如果有一个好一点的平台，那么VS的下场会很惨的，你VS三国是吧，扣扣现在有个英雄联盟，你觉得你VS三国还能活多久？VS平台是吧？扣扣的对战平台不给力，你先嚣张一短时间，时间到了，一样会很惨很惨。不知道用户在你心中是什么地位？狗日的VS对战平台。
