我使用的 WordPress 插件

2009-09-17 00:29

很大一部分用 WordPress 都喜欢写篇日志来分享在使用的插件，我也要赶个时髦，不能落伍。Ps.其实更大的原因就是为了除草。 下面简单的说下我所使用的 Wordpress 插件吧。

Akismet

这个不用过多的介绍了，一个强大的垃圾评论过滤工具。本人转到WP已经两个多月，收到过24条垃圾评论，其中3条是误杀， 误杀率暂为12.5%。

下载地址：这个WP都自带的，不给出来了。

WP-RecentComments

国内强人mg12开发的显示最新评论信息插件，自定义效果很强大。

下载地址：http://wordpress.org/extend/plugins/wp-recentcomments/

WP-MulticolLinks

还是国内强人人mg12开发的侧栏友情链接多列显示。

下载地址：http://wordpress.org/extend/plugins/wp-multicollinks/

WP-EasyArchives

怎么还是国内强人mg12的作品？这个插件用于自定义页面中文章存档的显示。

下载连接：http://wordpress.org/extend/plugins/wp-easyarchives/

WordPress Related Posts

更具当前日志的tag来生成出部分相关日志。

下载地址：http://fairyfish.net/2007/09/12/wordpress-23-related-posts-plugin/

No Self Pings

防止自己PING自己的文章。

下载地址：http://blogwaffe.com/2006/10/04/421/

Super Switch

关闭一些不必要的功能，比如关闭自动保存、不显示WP版本、关闭修订版等等。

下载地址：http://goto8848.net/projects/super-switch/

Wordpress Thread Comment

可嵌套回复评论，也能够发送邮件提醒评论者。

下载地址：http://blog.2i2j.com/plugins/wordpress-thread-comment

Google XML Sitemaps

这个不说了，看看Sitemaps应该都知道是什么东西。

下载地址：http://www.arnebrachhold.de/redir/sitemap-home/

MobilePress

博客的WAP版，可直接输入地址访问，不需要特别的路径，插件会自动判断是什么浏览器进行访问的。

下载地址：http://mobilepress.co.za/

wp slug translate

自动将日志标题翻译成英文或者拼音。

下载地址：http://blog.2i2j.com/

WP Kit CN

中文WordPress工具箱一直以来就是中文WP博客不可少的插件，而这款插件是中文WordPress工具箱的修改版。

下载地址：http://sexywp.com/wp-kit-cn.htm

WP-Syntax

格式化显示代码语法高亮插件，虽然我博客很少发代码，但是也有用得到的地方。

下载地址：http://wordpress.org/extend/plugins/wp-syntax/

Auto Highslide

一款图片特效的插件，效果大家随便找篇有图片的日志点击图片后就能看到。

下载地址：http://showfom.com/auto-hishslide-wordpress-plugin/

All in One SEO Pack

SEO插件，至今还没摸透怎么用，完全是默认设置。

下载地址：http://semperfiwebdesign.com/

就这么些了，本来还有两个插件的，刚刚想想还是不写了，因为我马上要把那两个插件干掉，感觉是多余的。。
