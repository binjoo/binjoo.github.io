请各位博友更新本博链接和博客名

2009-07-14 18:19

常来的应该都知道本博更换连接和博客名的事情了，这里不再多说了吧。

博客名称：冰剑博客
博客地址：jan.cm

另外给大家一个更新以前日志中带有本博旧链接的方法。首先进入后台->数据维护->Mysql，然后在输入框中输入如下SQL语句。

BO-BLOG的更新SQL语句：

update boblog_blogs set content = replace(content,'binjoo.cn','jan.cm')

WordPress的更新SQL语句：（感谢@威儿的提醒）

update wp_posts set post_content = replace(post_content,'binjoo.cn','jan.cm')

我自己已经用过了，绝对没问题，在后台MYSQL处执行就可以了，但是数据库操作有一定危险，请备份一下，放心一点。
