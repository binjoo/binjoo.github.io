不能落后，我也有“云”

2010-07-17 22:19

自从 Google 提出“云计算”的概念以来，各大 IT 厂商都加入了“云”的战争，最早的当然是流氓出身的360的“云杀毒”，慢慢的又有很多“云”军团加入了战争，比如“云存储”、“云安全”、“云输入法”等等。

所谓“云”，其实指的是后端（服务器端），平时我们很少能够看到的那一端，正因为平时难得看到，所以有一种虚无缥缈的感觉，也许就是因为这个原因，才被称为“云”吧。我们平时能够看到的是什么呢，当然是自己用的PC和手机这些东西了，也就是所谓的“客户端”。[引用百度云查杀]

去年，写了一篇关于添加搜狗云输入法的文章，今天是告诉大家把搜狗云、QQ云输入法都加入到博客里来。

第一步：添加 ime.js 文件

首先，把 ime.js 下载保存至服务器，或者把下列代码保存为 js 文件上传；

function ime_qq() {
    javascript: (function(q) { !! q ? q.toggle() : (function(d, j) {
            j = d.createElement('script');
            j.src = 'http://ime.qq.com/fcgi-bin/getjs';
            j.setAttribute('ime-cfg', 'lt=2');
            d.getElementsByTagName('head')[0].appendChild(j)
        })(document)
    })(window.QQWebIME)
};
function ime_sogou() {
    javascript: void((function() {
        var e = document.createElement('script');
        e.setAttribute('src', 'http://web.pinyin.sogou.com/web_ime/init.js');
        document.body.appendChild(e)
    })())
};

第二步：添加启动按钮

然后把下面这段代码贴到你要显示按钮的地方。 

<a href="javascript:void(0);" class="ime_qq" onclick="ime_qq">QQ拼音</a>
<a href="javascript:void(0);" class="ime_sogou" onclick="ime_sogou">搜狗拼音</a>

好了，大功告成，演示大家看评论框提交按钮边上那个键盘。
